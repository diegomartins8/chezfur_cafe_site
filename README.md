# chezfur_cafe_site
Multiple page website for a fictional cat cafe built with HTML and CSS. Features responsive layout, navigation, and image links.

---

### Live site  
https://diegomartins8.github.io/chezfur_cafe_site/ 

---

### Tools used
- HTML  
- CSS

--- 

### Description
This site was built for a Web Design course assignment, with the objective of showcasing layout, navigation, and styling using HTML and CSS.

The site entails of 4 pages that connect into one small experience. 
- Home page introduces the cafe with a sense of the environment.
- Menu page shows the basic food and drink options.
- Hosts page focuses on the main attraction of the cafe, and an image link to the fictitious site sponsor.
- Contact page encourages visitor to reach out via email or walk-in.

--- 

### Skills 
- Each page structure is built with HTML.
- All styling through a single CSS file, with consistent colors for the header, footer, and background.
- Images are in their own folder and sized to fit the layout.
- Pages connected through text links, and one image linked to another site.
